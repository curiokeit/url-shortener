A full-stack URL shortening system built with **ASP.NET Core Web API** and **MVC architecture**.

This project simulates a real-world SaaS backend system with authentication, link management, and a basic user interface.

---

## 🔥 Features

- 🔗 URL shortening service
- ✏️ Custom short links support
- ⏳ Expiration date for links
- 🔐 JWT-based authentication
- 👤 User registration & login
- 📊 Link management dashboard
- 🌐 RESTful API design
- 🖥️ Basic MVC frontend interface

---

## 🛠 Technologies

- ASP.NET Core Web API
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- JWT Authentication
- Bootstrap
